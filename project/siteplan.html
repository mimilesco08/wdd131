Website Planning Document

Site Name

Baseline Fundamentals

Domain: baseline-fundamentals.com (or baseline-hoops.org)

The name "Baseline Fundamentals" was chosen because it represents the foundation of basketball training. The "baseline" is a fundamental part of basketball court terminology, and "fundamentals" emphasizes our focus on core skills development. This name clearly communicates that we're about building strong foundational skills in basketball.

Logo Concept: A basketball with geometric lines representing the baseline of a court, combined with clean typography that emphasizes reliability and skill development.

Site Purpose

Baseline Fundamentals serves as a comprehensive digital training hub for basketball players of all skill levels who want to master the essential fundamentals of the game. The website provides:

  • Structured Skill Development: Organized drills and exercises for shooting, dribbling, defense, and other core basketball skills
  • Progress Tracking: Interactive practice logging system to help players monitor their improvement over time
  • Daily Training Guidance: Featured "Drill of the Day" to provide consistent, focused practice recommendations
  • Educational Resources: Clear explanations of fundamental techniques with step-by-step instructions
  • Motivational Support: Tools and content designed to encourage consistent practice and skill development

The site bridges the gap between casual basketball play and structured skill development, making professional-level training techniques accessible to everyone.

Target Audience Scenarios

Scenario 1 - High School Player:
"I'm on my high school basketball team but I struggle with my free throw shooting. What specific drills can help me improve my shooting consistency, and how should I track my progress?"
Scenario 2 - Beginner Adult:
"I'm 25 and just starting to learn basketball. What are the most important fundamental skills I should focus on first, and how do I practice them correctly?"
Scenario 3 - Youth Coach:
"I coach 12-year-olds and need simple, effective drills that will help them develop proper dribbling technique. What drills work best for this age group?"
Scenario 4 - Parent:
"My daughter loves basketball and wants to practice at home. What fundamental skills can she work on in our backyard, and how can I help track her improvement?"

Typography

Primary Font: Roboto Slab (Google Fonts)

Usage: Main headings (H1, H2, H3), form legends, emphasized text

Characteristics: A modern slab serif that combines readability with athletic confidence. The slab serifs give it a strong, grounded feel perfect for sports branding.

Secondary Font: Roboto (Google Fonts)

Usage: Body text, navigation, form inputs, descriptions, captions

Characteristics: Clean, highly legible sans-serif designed for optimal readability across all devices. Excellent for user interfaces and body content.

Font Hierarchy & Implementation:
  • Google Fonts Import: @import url('https://fonts.googleapis.com/css2?family=Roboto+Slab:wght@400;700&family=Roboto:wght@400;500&display=swap');
  • H1: 2rem, Roboto Slab 700, White on dark backgrounds
  • H2, H3: Roboto Slab 700, Basketball Red (#CE1141)
  • Body: 1rem, Roboto 400, Dark text (#333)
  • Navigation: Roboto 500, Responsive hover states

Content Strategy

Site Pages:

  1. Home (index.html): Welcome, daily drill, navigation to main features
  2. Skills & Drills (skills.html): Categorized library of basketball drills
  3. Log Your Practice (log.html): Practice session tracking and progress monitoring
  4. References (references.html): Content sources and media credits

Key Content Elements:

  • Step-by-step drill instructions with difficulty levels
  • Progress tracking forms and visual feedback
  • Motivational messaging to encourage consistent practice
  • Clear navigation and intuitive user flow

Technical Implementation

File Structure:

  • index.html - Home page with hero section and drill of the day
  • skills.html - Interactive drill library with filtering
  • log.html - Practice logging system with local storage
  • references.html - Content and media attribution
  • css/styles.css - Comprehensive responsive stylesheet
  • js/scripts.js - Interactive functionality and data management

Key Technical Features:

  • Mobile-First Responsive Design: CSS Grid and Flexbox layouts
  • CSS Custom Properties: Centralized theming with CSS variables
  • JavaScript Data Management: Drill filtering and localStorage for practice logs
  • Interactive Elements: Dynamic drill rotation, form handling, skill filtering
  • Google Fonts Integration: Roboto Slab and Roboto font families
  • Accessibility Features: Semantic HTML, proper form labels, keyboard navigation
  • Performance Optimized: Efficient DOM manipulation and event handling

JavaScript Functionality:

  • Drill of the Day: Random drill selection from predefined array
  • Skill Filtering: Dynamic content filtering on Skills page
  • Practice Logging: Form submission with localStorage persistence
  • DOM Management: Template literals for dynamic HTML generation
  • Date Handling: Automatic footer updates with current year and last modified date

Data Structure:

The website uses a structured JavaScript array to store drill information:

  • Skills Categories: Shooting, Dribbling, Passing, Defense
  • Drill Properties: skill, name, description
  • Practice Logs: date, skill, duration, notes (stored in localStorage)
